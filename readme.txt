1. Project Name : cucumberProject
2. Introduction : This project is to select fastest and cheapest flights between the selected locations.
3. Tools used 	: Eclipse IDE, Build Tool - Maven and Cucumber Framework.
4. Steps to import the project : Click on the file, Select import and give the path of the project and click on finish.
5. How to Run the project : It can be executed from Runner class.Right click on the file and select "Run As" then "JUnit Test".
6. How and where to view the reports : Default cucumber reports(target/cucumber-reports/testCaseResults.html/index.html).
